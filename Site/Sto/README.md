# АвтоМастер Про — Премиальный автосервис

## 🚗 Описание проекта

Современный веб-сайт для премиального автосервиса с акцентом на прозрачности, технологичности и доверии. Разработан с использованием React и современных технологий анимации.

## ✨ Особенности

- **Кинематографические анимации** — GSAP + Framer Motion для плавных переходов
- **Интерактивные элементы** — калькулятор стоимости, форма записи с шагами
- **Эффект масляных капель** — уникальный визуальный элемент при скролле
- **Адаптивный дизайн** — оптимизирован для всех устройств
- **Премиальный UI** — индустриальная тёмная тема с акцентными цветами

## 🛠 Технологии

- **React 18** — современный UI фреймворк
- **Vite** — быстрая сборка и HMR
- **Framer Motion** — анимации и transitions
- **Lucide React** — иконки
- **React Router** — маршрутизация
- **CSS Modules** — стилизация компонентов

## 📦 Установка

```bash
# Клонировать репозиторий
cd СТО

# Установить зависимости
npm install

# Запустить в режиме разработки
npm run dev

# Сборка для продакшена
npm run build
```
